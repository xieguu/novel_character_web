# 小说人物提取与模板生成器 (Novel Character Web)

一个基于 AI 的小说人物管理系统，支持从小说文本中自动提取人物信息、生成人物档案、创建人物关系网络、导出 PDF 等功能。

**项目地址**：https://novelchar-khc3tder.manus.space

## 🎯 项目概述

本项目是一个全栈 Web 应用，旨在帮助小说作者和文学爱好者：

- **自动提取人物信息** - 使用 LLM 从小说文本中智能识别人物及其特征
- **管理人物档案** - 创建、编辑、删除人物档案，支持详细的人物信息记录
- **可视化关系网络** - 展示人物之间的关系，支持手动编辑
- **生成人物头像** - 使用 AI 图像生成服务为人物创建头像
- **导出与分享** - 支持 PDF 导出、Markdown 导出、项目协作分享

## ⚠️ 项目弊端与已知问题

### 1. LLM 配额限制 ⛔
**问题**：人物提取功能依赖 LLM API，当账户配额耗尽时无法进行提取
- **影响**：用户无法上传文本进行人物提取
- **解决方案**：
  - 升级 Manus 订阅或充值积分
  - 考虑添加本地 NLP 模型作为备选方案
  - 实现配额检查和友好的提示信息

### 2. 协作编辑功能不完整 🔄
**问题**：虽然数据库表已创建，但后端 API 和前端界面还未完全实现
- **已完成**：数据库表结构、权限验证中间件、分享链接生成
- **待完成**：
  - 协作者管理 API（添加/移除/修改权限）
  - 项目共享前端界面
  - 权限检查在实际操作中的应用
- **影响**：多用户协作功能暂不可用

### 3. 实时同步缺失 ⏳
**问题**：WebSocket 实时同步功能未实现
- **影响**：多用户同时编辑时无法实时同步数据
- **风险**：可能出现数据冲突和覆盖
- **解决方案**：需要实现 WebSocket 服务器和前端实时更新逻辑

### 4. 版本恢复功能缺失 📝
**问题**：编辑历史已记录，但不支持恢复到历史版本
- **影响**：用户无法撤销误操作
- **解决方案**：实现版本恢复 API 和前端 UI

### 5. PDF 导出依赖问题 📄
**问题**：PDF 导出功能依赖 `pdf-lib` 库，但在某些环境下可能出现兼容性问题
- **影响**：PDF 导出可能失败
- **已知错误**：之前出现过 `node-fetch` 导入错误
- **解决方案**：已修复，但需要在生产环境中验证

### 6. 搜索性能优化缺失 🔍
**问题**：高级搜索功能已实现，但未进行性能优化
- **影响**：大数据量下搜索可能较慢
- **解决方案**：
  - 添加数据库索引
  - 实现搜索结果缓存
  - 使用全文搜索引擎（如 Elasticsearch）

### 7. 自定义模板保存缺失 💾
**问题**：用户无法将创建的人物档案保存为自己的模板
- **影响**：只能使用预设模板
- **解决方案**：实现用户模板保存和管理功能

### 8. 错误处理不够完善 ❌
**问题**：某些错误信息不够详细，难以定位问题
- **已改进**：提取错误现在返回更详细的错误信息
- **待改进**：其他 API 的错误处理

├── routers.ts                    # 所有 tRPC 路由定义
├── db.ts                         # 数据库查询助手
├── characterExtractor.ts         # LLM 人物提取逻辑
├── randomCharacterGenerator.ts   # 随机人物生成
├── avatarGenerator.ts            # AI 头像生成
├── pdfExporter.ts                # PDF 导出
├── collaborationService.ts       # 协作编辑服务
├── templateService.ts            # 人物模板库服务
├── fileUploadHandler.ts          # 文件上传处理
└── _core/                        # 核心基础设施
    ├── llm.ts                    # LLM 调用接口
    ├── context.ts                # tRPC 上下文
    ├── trpc.ts                   # tRPC 配置
    └── ...
```

### 前端结构
```
client/src/
├── pages/                        # 页面组件
│   ├── Home.tsx                  # 首页
│   ├── NovelDetail.tsx           # 项目详情页

├── components/                   # 可复用组件
